# GSD 路线图

## 阶段一：生产底座补强

Bead：`codex-hvo7`

目标：建立可持续开发和真实试用基础。包括持久化、repository/service 分层、安全登录、权限/额度、API 校验、统一错误和基础测试。

验收：重启服务后关键业务数据不丢失，登录逻辑不再依赖明文密码，`npm run check` 通过。

## 阶段二：AI 员工任务中心与审批闭环

Bead：`codex-719u`

目标：新增任务中心、AI 员工控制台、执行日志和审批中心，形成“目标 -> 拆解 -> 工具调用 -> 审批 -> 结果”的闭环。

## 阶段三：抖音真实平台闭环强化

Bead：`codex-sf5k`

目标：优先打通抖音真实授权、同步、失败记录、能力矩阵和待发送队列。

## 阶段四：知识库与智能客服升级

Bead：`codex-02hg`

目标：将客服从模板回复升级为结构化知识库、RAG、依据引用、风险识别和人工升级。

## 阶段五：AI 视频能力产品化

Bead：`codex-9qqh`

目标：建立素材库、品牌规范、批量生成、异步队列、模型能力矩阵、成本统计和合规检查。

## 阶段六：经营分析与 ROI 闭环

Bead：`codex-qsa6`

目标：将 AI 任务与内容、流量、客服、成交、售后和复购结果关联，形成 ROI 分析。
